# javascript-web-text-to-pdf

## Description
A small project to demonstrate a javascript
library that turns text to pdf.

## Tech stack
- jquary 3.5
- jspdf

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
